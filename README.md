TalanT - launcher app.
# Management.
A - go to the previous program;
D - go to the next program;
E - Select an application.
# You can change sprites.
<img width="1202" height="295" alt="2026-6-1 14-12-28" src="https://github.com/user-attachments/assets/77c1137e-ad2b-47a8-8a15-373f2954e67b" />
There are sprites in the app folder:
"Gayka1",
"Gayka45", 
"Gayka90", 
"Gayka135", 
"Gayka180". 
You can change the sprites with the same name and size. After that, the images will appear in the app.
<h1>You can also change the slot for the application.</h1>
<img width="227" height="266" alt="2026-6-1 14-17-24" src="https://github.com/user-attachments/assets/2beedfd9-1e76-4879-849d-3fa69caf58c9" />
In the application folder, there is a file called "Config1.txt". You can change the path, and the program will open the program specified in this file.
<h1>IMPORTANT!</h1>
It is recommended to specify the path in the following format: "C:\\...\\...\\...".
